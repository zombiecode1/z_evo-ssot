# 🧟 ZombieCoder

**যেখানে হারানো স্বপ্নগুলো আবার বাঁচে ওঠে**

> "সমাজে এবং পরিবারে এবং কাজের জন্য ব্যবহার করা প্রতিটি টুলের কাছ থেকে যখন তার নিজের অজান্তে যন্ত্রণা পেতে পেতে জম্বি হয়ে গিয়েছে কিন্তু অদম্য চিন্তা এবং এই জম্বি হওয়ার কষ্ট তাকে ভুলতে দেয়নি।"

---

## 💔 নামের পেছনের গল্প

"জম্বি" শব্দটি এখানে মৃতদেহকে বোঝায় না। এটি প্রতিনিধিত্ব করে কোটি কোটি মানুষের:

- 💔 যাদের **স্বপ্ন আছে** কিন্তু **সম্পদ নেই**
- 🎯 যাদের **দক্ষতা আছে** কিন্তু **স্বীকৃতি নেই**
- 😔 যারা ধীরে ধীরে **"জম্বি" হয়ে যাচ্ছে** — মানসিকভাবে হারিয়ে গিয়ে
- 🌟 যারা এখনও বিশ্বাস করে: **"আমি চেষ্টা করতে পারি, এমনকি হেরে গেলেও"**

**ZombieCoder** তাদের জন্য যারা হার মানতে রাজি নয়।

## 🌟 বাস্তব জীবনের গল্প

### রাশেদ — একজন ফ্রিল্যান্সার (ঢাকা)

> "আমি ৩ বছর ধরে ফ্রিল্যান্সিং করছি। প্রতিদিন রাত ২টা পর্যন্ত কাজ করি। কিন্তু ক্লায়েন্টরা বলে — 'আপনার কাজ ভালো না', 'আমি আরেকজনকে দিয়ে করাবো।' একদিন রাতে কাজ করতে করতে চোখে জল এসে গেল। তখনই আমি বুঝলাম — আমি ধীরে ধীরে 'জম্বি' হয়ে যাচ্ছি। তারপর আমি ZombieCoder পেলাম। এটি আমাকে বলল — 'আপনি যদি চেষ্টা করতে থাকেন, আমি আপনার পাশে আছি।' এখন আমি প্রতিদিন নতুন কিছু শিখছি। আমি আর 'জম্বি' নই।"

### নাদিয়া — একজন স্টুডেন্ট (চট্টগ্রাম)

> "আমি কম্পিউটার সায়েন্স পড়ছি। প্রোগ্রামিং শিখতে গিয়ে অনেক বার হাল ছেড়ে দিয়েছি। শিক্ষকরা বলেন — 'তোমার মতো ছাত্রদের জন্য সময় নেই।' বন্ধুরা বলে — 'তুই পারবি না।' একদিন ZombieCoder ব্যবহার করলাম। এটি আমাকে ধাপে ধাপে বুঝিয়ে দিল। কোনো বিচার নেই, কোনো তিরস্কার নেই। শুধু ধৈর্য ধরে বোঝানো। আজ আমি আমার প্রথম ওয়েবসাইট বানিয়েছি। আমি পারলাম।"

### কামাল — একজন ছোট ব্যবসায়ী (সিলেট)

> "আমার একটি ছোট দোকান। অনলাইনে বিক্রি করতে চাই কিন্তু জানি না কিভাবে। মেয়েকে বললাম — 'বাবা, আমাকে একটা ওয়েবসাইট বানিয়ে দাও।' মেয়ে বলল — 'বাবা, আমি পারব না।' তারপর একজন বন্ধু বলল — 'ZombieCoder আছে, সে সাহায্য করবে।' আমি বিশ্বাস করলাম না। কিন্তু চেষ্টা করলাম। আজ আমার নিজের ওয়েবসাইট আছে। আমি অনলাইনে বিক্রি করছি। আমার মেয়ে এখন গর্বিত — 'বাবা তুমি পারলে!'"

## 🔍 "লোকাল-ফার্স্ট" — সত্যি কথা

> "লোকাল বলতে দুনিয়ায় কিছুই নেই। আপনার কম্পিউটারের মাদারবোর্ড Intel, গ্রাফিক্স কার্ড NVIDIA, অপারেটিং সিস্টেম Windows, এডিটর VS Code, GitHub, ব্রাউজার Google Chrome — এগুলো ব্যতীত একজন মানুষ সভ্য সমাজে এখন থাকতে পারে না।"

**আমরা মিথ্যা বলি না।** "লোকাল-ফার্স্ট" দিয়ে আমরা বোঝাই:

| বিষয় | সত্যি কথা |
|-------|-----------|
| AI মডেল | আপনার কম্পিউটারেই চলে (লোকাল LLM) |
| ডাটা স্টোরেজ | আপনার হার্ড ডিস্কে থাকে |
| কনভারসেশন | SQLite ফাইলে আপনার কম্পিউটারে সেভ হয় |
| নেটওয়ার্ক | ইন্টারনেট ছাড়াও কাজ করে (লোকাল মডেল) |
| কোস্ট | সম্পূর্ণ বিনামূল্যে |

**আমরা যা স্বীকার করি:**
- ❌ সম্পূর্ণ লোকাল — এটি সম্ভব নয়
- ❌ কোনো ডাটা বাইরে যায় না — এটি মিথ্যা
- ✅ আপনি আপনার ডাটার উপর সর্বোচ্চ নিয়ন্ত্রণ রাখতে পারেন

বিস্তারিত দেখুন: [Local First — সত্যি কথা](docs/local-first.html)

## 🚀 দ্রুত শুরু

### পূর্বশর্ত
- Node.js >= 18.0.0
- npm বা yarn

### ইনস্টলেশন

```bash
# রিপোজিটরি ক্লোন করুন
git clone https://github.com/zombiecode1/Ssot.git
cd Ssot

# ডিপেন্ডেন্সি ইনস্টল করুন
npm install

# বিল্ড করুন
npm run build

# সার্ভার চালু করুন
npm start
```

বস! এখন `http://localhost:9999` এ গিয়ে ZombieCoder এর সাথে কথা বলুন।

## 📊 SSOT (সিঙ্গেল সোর্স অব ট্রুথ)

SSOT.md হলো অটো-জেনারেটেড ডকুমেন্টেশন যেটি:

- ✅ আপনার প্রজেক্ট স্ট্রাকচার স্ক্যান করে
- ✅ কী ফাইল এবং কনফিগারেশন ডকুমেন্ট করে
- ✅ ফাইল পরিবর্তন হলে অটোমেটিক্যালি আপডেট হয়
- ✅ সব এজেন্টের জন্য কনটেক্স্ট হিসেবে কাজ করে

### SSOT ফ্ল্যাগ সিস্টেম

| ফ্ল্যাগ | অবস্থা | বিবরণ |
|--------|--------|--------|
| ০ | স্ক্যান হয়নি | ডিরেক্টরি কখনো ইনডেক্স হয়নি |
| ১ | স্ক্যান হচ্ছে | বর্তমানে প্রসেস হচ্ছে |
| ২ | ইনডেক্সড | SSOT আপ-টু-ডেট |
| ৩ | ত্রুটি | স্ক্যান ব্যর্থ |
