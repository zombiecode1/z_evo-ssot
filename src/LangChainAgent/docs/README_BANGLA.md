# ZombieCoder Agent - সহজ বাংলা গাইড

## সাধারণ মানুষের জন্য

এই ডকুমেন্টটি техনিক্যাল জ্ঞান ছাড়াই ZombieCoder Agent কিভাবে কাজ করে তা বুঝতে সাহায্য করবে।

---

## 🎯 ZombieCoder আসলে কী?

ZombieCoder হলো একটি **বুদ্ধিমান সফটওয়্যার প্যাকেট** যা আপনার যেকোনো প্রজেক্টের সাথে যুক্ত করে ব্যবহার করা যায়। এটি মূলত ৫টি আলাদা আলাদা বিশেষজ্ঞ এআই (AI) দিয়ে তৈরি, যারা各自 নিজ নিজ কাজে দক্ষ।

### সহজ উদাহরণ

মনে করুন আপনার একটি রেস্টুরেন্ট আছে। ZombieCoder হলো আপনার ৫ জন বিশেষজ্ঞ কর্মচারী:

1. **স্থপতি (Architect)** - রেস্তোরাঁর নকশা করে
2. **রান্নাওয়ালা (Engineer)** - খাবার তৈরি করে
3. **টেস্টার (QA)** - খাবারের স্বাদ পরীক্ষা করে
4. **লেখক (Documentation)** - মেনু ও নির্দেশিকা লেখে
5. **ম্যানেজার (Operations)** - সব কিছু পরিচালনা করে

---

## 📦 এটি কিভাবে পাবেন?

### ধাপ ১: ফাইল নামানো

আপনার প্রজেক্টের ফোল্ডারে `LangChainAgent` নামের ফোল্ডারটি কপি করুন।

### ধাপ ২: ইন্সটল করা

টার্মিনাল বা কমান্ড প্রম্পটে গিয়ে লিখুন:

```bash
cd LangChainAgent
npm install
```

### ধাপ ৩: সেটআপ করা

```bash
cp .env.example .env
```

এরপর `.env` ফাইলটি ওপেন করে আপনার প্রয়োজনীয় তথ্য দিন।

### ধাপ ৪: চালু করা

```bash
npm run build
```

ব্যাস! এখন এটি ব্যবহারের জন্য প্রস্তুত।

## 🤖 ৫ জন বিশেষজ্ঞের পরিচয়

### ১. স্থপতি (Solution Architect)

**কি করে:**
- বড় প্রজেক্টের পরিকল্পনা করে
- কোন প্রযুক্তি ব্যবহার করতে হবে তা ঠিক করে
- ঝুঁকি চিহ্নিত করে

**কখন ব্যবহার করবেন:**
- নতুন প্রজেক্ট শুরু করার সময়
- সিস্টেম ডিজাইন করার সময়
- প্রযুক্তি নির্বাচনের সময়

**উদাহরণ:**
```
"আমার একটি ই-কমার্স ওয়েবসাইট বানাতে হবে যেখানে দিনে ১ লাখ মানুষ কিনা করতে পারবে।"
```

**কি করে:**
- কোড লেখে

- পুরনো কোড উন্নত করে

**কখন ব্যবহার করবেন:**

- কোনো সমস্যা ঠিক করার সময়

**কি করে:**
- টেস্ট প্ল্যান তৈরি করে
- বাগ খুঁজে বের করে
- কোয়ালিটি রিপোর্ট দেয়

**কখন ব্যবহার করবেন:**
- নতুন ফিচার টেস্ট করার সময়
- বাগ বিশ্লেষণ করার সময়
- কোয়ালিটি চেক করার সময়

**উদাহরণ:**
```
"লগইন সিস্টেমের জন্য টেস্ট প্ল্যান তৈরি করে দাও।"
```

**কি করে:**
- ডকুমেন্ট লেখে

- ব্যবহারকারীর ম্যানুয়াল লেখে

**কখন ব্যবহার করবেন:**

- বিপদের সময় করণীয় ঠিক করে

**কখন ব্যবহার করবেন:**

## 💡 কিভাবে ব্যবহার করবেন?

### সাধারণ ব্যবহার

```javascript
// ZombieCoder চালু করা
const agent = new LangChainAgent();

// স্থপতির সাহায্য নেওয়া
const plan = await agent.solutionArchitect.analyze({
  requirements: "একটি অনলাইন শপ বানাতে চাই"
});

// ডেভেলপারের সাহায্য নেওয়া
const code = await agent.developmentEngineer.developFeature({
  description: "ইমেইল ভ্যালিডেশন ফাংশন"
});

// টেস্টারের সাহায্য নেওয়া
const testPlan = await agent.qualityAssurance.createTestPlan({
  featureDescription: "লগইন সিস্টেম"
});

// লেখকের সাহায্য নেওয়া
const docs = await agent.documentation.createAPIDocumentation({
  apiName: "User API",
  endpoints: [...]
});

// অপারেটরের সাহায্য নেওয়া
const deployPlan = await agent.operations.planDeployment({
  applicationName: "MyApp",
  environment: "production"
});
```
