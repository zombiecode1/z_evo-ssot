পরিচিতি

ZombieCoder Agent Module হলো একটি প্লাগ-অ্যান্ড-প্লে AI এজেন্ট প্যাকেজ যা যেকোনো প্রজেক্টের সাথে সহজেই ইন্টিগ্রেট করা যায়। এটি ৫টি বিশেষজ্ঞ AI এজেন্ট নিয়ে তৈরি, যারা সফটওয়্যার ডেভেলপমেন্টের বিভিন্ন পর্যায়ে কাজ করে।

এই রিপোর্টে আমরা দেখবো কিভাবে এই এজেন্ট সিস্টেমটি আমাদের Proxi Bridge Server (http://localhost:9999) এর সাথে ইন্টিগ্রেট করা যায়।

ইন্টিগ্রেশন পরিকল্পনা

ZombieCoder Agent কে Proxi Bridge এর সাথে ইন্টিগ্রেট করার জন্য নিচের ধাপগুলো অনুসরণ করতে হবে:

ধাপ ১: Proxi Bridge এর Agent Endpoint এর সাথে সংযোগ

বিদ্যমান Proxi Bridge এ POST /v1/agent/chat endpoint আছে। ZombieCoder এর প্রতিটি এজেন্টকে এই endpoint এর মাধ্যমে route করা যায়।

// Proxi Bridge endpoint POST http://localhost:9999/v1/agent/chat { "model": "mimo-v2.5-free", "messages": [{"role": "user", "content": "..."}], "agent_type": "solution_architect" // নতুন agent_type প্যারামিটার }

ধাপ ২: UniversalLLMController কে Proxi Bridge এ রুট করা

ZombieCoder এর UniversalLLMController কে পরিবর্তন করে Proxi Bridge কে fallback হিসেবে ব্যবহার করা যায়:

// UniversalLLMController.ts - পরিবর্তিত সংস্করণ const proxiClient = createOpenAI({ baseURL: "http://localhost:9999/v1", // Proxi Bridge apiKey: "local-proxi", }); // Fallback chain: // 1. OpenCode Zen (Primary) // 2. Proxi Bridge (Secondary) // 3. Ollama (Tertiary)

ধাপ ৩: Identity Anchoring একত্রীকরণ

দুটো সিস্টেমেই identity anchoring আছে। Proxi Bridge এ system prompt দিয়ে, ZombieCoder এ identity.json দিয়ে। এদের একত্র করতে হবে:

// Proxi Bridge এর SSOT.md তে ZombieCoder identity যোগ করা // + ZombieCoder এর IdentityService থেকে Proxi Bridge এ পাঠানো

ধাপ ৪: নতুন Agent Chat API তৈরি

Proxi Bridge তৈরি করুক এমন একটি endpoint যেখানে agent_type অনুযায়ী সঠিক ZombieCoder agent নির্বাচিত হবে:

// নতুন endpoint: /v1/agent/specialist POST http://localhost:9999/v1/agent/specialist { "agent_type": "development_engineer", "task": "fixBug", "params": { "bugDescription": "Login form validation error", "errorCode": "ERR_VALIDATION_001", "errorMessage": "Email field accepts invalid format" } }

ধাপ ৫: RAG সিস্টেম ইন্টিগ্রেশন

Proxi Bridge এর SSOT.md based RAG সিস্টেমকে ZombieCoder এর agent গুলোর সাথে যুক্ত করতে হবে, যাতে প্রজেক্টের ডকুমেন্টেশন থেকে context নিয়ে কাজ করতে পারে।

চূড়ান্ত মতামত

ZombieCoder Agent System এর কোড অত্যন্ত মানসম্মত ও অথেন্টিক। এটি একটি ভালোভাবে পরিকল্পিত, মডুলার এবং স্কেলেবল সিস্টেম।

Proxi Bridge এর সাথে ইন্টিগ্রেশনের জন্য:

  • UniversalLLMController কে Proxi Bridge endpoint এ রুট করুন
  • Proxi Bridge এর agent/chat endpoint এ agent_type parameter যোগ করুন
  • দুই সিস্টেমের identity anchoring একত্র করুন
  • ZombieCoder এর agents কে Proxi Bridge এর MawlanaRouter দিয়ে route করুন
  • RAG সিস্টেম Proxi Bridge এর SSOT.md থেকে পরিচালিত করুন
  • একটি unified agent gateway API তৈরি করুন

এই ইন্টিগ্রেশনের ফলে একটি শক্তিশালী, বহু-ভাষিক, স্বয়ংক্রিয় AI এজেন্ট সিস্টেম তৈরি হবে যা বাংলাদেশের যেকোনো ডেভেলপারের জন্য অত্যন্ত কার্যকর হবে।
