📌 এটি কী?

ZombieCoder একটি মাল্টি-এজেন্ট AI সিস্টেম যা ৫টি ভিন্ন AI এজেন্ট নিয়ে কাজ করে। প্রতিটি এজেন্ট আলাদা আলাদা কাজে বিশেষজ্ঞ — যেমন কোড লেখা, আর্কিটেকচার ডিজাইন, টেস্টিং, ডকুমেন্টেশন, এবং অপারেশন।

এটি সম্পূর্ণরূপে কনফিগারেবল — কোন এজেন্ট কোন মডেল, কোন প্রোভাইডার, এবং কোন URL ব্যবহার করবে তা .env ফাইলের মাধ্যমে সেট করা যায়। কোডে কোনো হার্ডকোডেড মডেল নেই।

🏗️ আর্কিটেকচার

সিস্টেমটি তিনটি লেয়ারে বিভক্ত:

  1. LLM Gateway — বিভিন্ন AI প্রোভাইডার (OpenCode Zen, Ollama, Gemini, ইত্যাদি) ম্যানেজ করে। এনভায়রনমেন্ট ভেরিয়েবল থেকে প্রোভাইডার ও মডেল সিলেক্ট করে।
  2. Agent Layer — ৫টি স্পেশালিস্ট এজেন্ট, প্রত্যেকের নিজস্ব system prompt এবং কনফিগারেশন। BaseAgent → AgentLoop → specialist agent।
  3. Orchestrator — MultiAgentOrchestrator এজেন্টদের মধ্যে কাজ ভাগ করে এবং রাউটিং করে।

সার্ভার রান করে Express.js-এর উপর (port 3000), এবং POST /api/chat এন্ডপয়েন্টের মাধ্যমে সব এজেন্টের সাথে যোগাযোগ করা যায়।

🔧 কনফিগারেশন

সব কনফিগারেশন .env ফাইল থেকে লোড হয়। নিচের ভেরিয়েবলগুলো ব্যবহার করে প্রতিটি এজেন্টের প্রোভাইডার, URL, API key, এবং মডেল আলাদাভাবে সেট করা যায়:

Env VariableDescriptionExample
LLM_PROVIDERDefault provider (opencode / universal)opencode
LLM_BASE_URLDefault base URLhttps://opencode.ai/zen/v1
LLM_API_KEYDefault API keysk-xxx
LLM_MODELDefault modeldeepseek-v4-flash-free
AGENT_<ROLE>_PROVIDERPer-agent provider overrideAGENT_QUALITY_ASSURANCE_PROVIDER=opencode
AGENT_<ROLE>_BASE_URLPer-agent URL overrideAGENT_ARCHITECT_BASE_URL=http://localhost:5010/v1
AGENT_<ROLE>_API_KEYPer-agent API key overrideAGENT_OPS_API_KEY=key-xxx
AGENT_<ROLE>_MODELPer-agent model overrideAGENT_DEVELOPMENT_ENGINEER_MODEL=north-mini-code-free

<ROLE> হলো এজেন্টের role নাম uppercase এবং underscores-এ: SOLUTION_ARCHITECT, DEVELOPMENT_ENGINEER, QUALITY_ASSURANCE, DOCUMENTATION, OPERATIONS।

POST /api/chat

curl -X POST http://localhost:3000/api/chat \
  -H 'Content-Type: application/json' \
  -d '{"message":"Hello","agentType":"qa"}'

🔌 প্রোভাইডার পরিবর্তন

.env ফাইল এডিট করলেই যেকোনো এজেন্টের প্রোভাইডার পরিবর্তন করা যায়। উদাহরণ:
