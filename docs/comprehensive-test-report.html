📂 ফোল্ডার স্ট্রাকচার

src/ — কোর সোর্স

  • src/index.ts — মেইন এন্ট্রি পয়েন্ট (Express server)
  • src/config/ — Persona কনফিগারেশন
  • src/types/ — TypeScript টাইপ ডিফিনিশন
  • src/routes/ — API রুট ডিফিনিশন (৩০+ এন্ডপয়েন্ট)
  • src/controllers/ — ৫টি কন্ট্রোলার (OpenAI, Agent, MCP, Zombie, DB)
  • src/services/ — ১৯টি সার্ভিস (Groq, Pipeline, RAG, LangChain, SSE Event Broadcaster, ইত্যাদি)
  • src/middleware/ — ৩টি মিডলওয়্যার (Auth, Identity, Logging)
  • src/providers/ — AI SDK রেজিস্ট্রি ও প্রাইসিং
  • src/tools/ — LangChain টুলস (read/write/list/run)
  • src/mcp/ — MCP সার্ভার ও ক্লায়েন্ট
  • src/adapter/ — Streaming অ্যাডাপ্টার (SSE, পোর্ট ৩৩৩৩)
  • src/admin/ — অ্যাডমিন প্যানেল কন্ট্রোলার
  • src/wrapper/ — OpenAI-কম্প্যাটিবল র্যাপার
  • src/LangChainAgent/ — ৫ স্পেশালিস্ট এজেন্ট + টুলস + মেমরি

অন্যান্য ডিরেক্টরি

  • dist/ — কম্পাইল্ড আউটপুট (১৪টি ডিরেক্টরি)
  • test/ — ৩টি টেস্ট ফাইল + JSON রিপোর্ট + ডকুমেন্টেশন
  • scripts/ — ৭টি ইউটিলিটি স্ক্রিপ্ট
  • docs/ — ৬টি ডকুমেন্টেশন ফাইল (HTML + MD)
  • transport/ — সার্ভিস ম্যানেজার, WS সার্ভার, স্ট্যাটাস HTML
  • mcp/ — ৫টি এডিটর MCP কনফিগ (VS Code, JetBrains, Zed, Windsurf)
  • .opencode/ — OpenCode এডিটর কনফিগ (১৪ মডেল, ২ এজেন্ট)
  • .zombiecoder/ — SSOT.md, কানেক্টেড ক্লায়েন্ট, রানটাইম JSON
